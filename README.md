# Conference Event Planner

Conference Event Planner is a React budgeting app for planning the cost of a conference or business event. Users can choose venue rooms, add event equipment, select meal options, enter the number of attendees, and see the total event cost update automatically.

## Features

- Select conference venues and adjust room quantities
- Add event equipment such as projectors, speakers, microphones, whiteboards, and signage
- Choose meal options for event attendees
- Enter the number of people attending the event
- View a detailed cost breakdown for venues, add-ons, meals, and the final total
- Manage shared app state with Redux Toolkit

## Tech Stack

- React
- Vite
- Redux Toolkit
- React Redux
- CSS

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## About

This project was built to demonstrate my knowledge of React and show Redux library usage in action. It highlights component-based UI development, centralized state management, dynamic rendering, and real-time cost calculations based on user selections.
